Telert API

Welcome to the Telert API server. This server provides endpoints to send notifications to various messaging services including Telegram, Teams, Slack, Discord, Pushover, and more.

Available Endpoints

GET /

Returns a status message indicating the API is running.

GET /status

Returns the current configuration status and information about configured providers.

GET /providers

Lists all configured notification providers and identifies which ones are set as defaults.

POST /send

Sends a notification to the configured providers.

Example request body:

{
  "message": "Hello from Telert!",
  "provider": "telegram",   // Optional: specific provider or list of providers
  "all_providers": false    // Optional: send to all configured providers
}
GET /health

Health check endpoint for container monitoring.
